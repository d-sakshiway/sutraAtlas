Creating Collections
  • Use descriptive names (e.g., "Machine Learning 2025", "Web Development Resources")
  • Add detailed descriptions to remember the purpose
  • Organize by subject, project, or learning path
  • Consider creating separate collections for different skill levels
Managing Collections
  • Edit collection names and descriptions anytime
  • Delete empty collections to keep things organized
  • Use the search feature to find specific collections quickly
  • Sort collections by name or creation date

Adding Resources
  • Title: Use clear, searchable titles
  • Authors: Include author names for books/articles
  • URL: Add direct links to online resources
  • Status: Set initial status (usually "Not Started")
Pro Tip: URLs are automatically formatted - you can enter "example.com" and it becomes "https://example.com"
Managing Resources
  • Update status as you progress through materials
  • Edit titles and authors to improve searchability
  • Add URLs later if you find online versions
  • Delete resources that are no longer relevant
Note: Deleting resources is permanent - consider changing status to "Not Relevant" instead

Status Meanings
Not Started Resources you plan to use but haven't begun
In Progress Currently reading/studying this resource
Paused Temporarily stopped, plan to resume later
Completed Finished reading/studying this resource
Reference Keep for future reference or lookup
Best Practices
  • Update regularly: Change status as you progress
  • Use "Paused" wisely: For resources you intend to resume
  • "Reference" status: For documentation, manuals, lookup resources
  • Review "Completed": Periodically review finished resources
Achievement Tip: Use search filters to see all "Completed" resources for motivation!

Search Features
  • Collection Search: Find collections by name or description
  • Resource Search: Search titles, authors, or content
  • Status Filtering: View resources by completion status
  • Quick Filters: Use predefined filters for common searches
Organization Tips
  • Create collections by topic, project, or difficulty level
  • Use consistent naming conventions for easier searching
  • Regularly review and clean up outdated resources
  • Group related resources in the same collection

Profile Management
  • Update username anytime (3-50 characters)
  • Username can contain letters, numbers, hyphens, underscores
  • Email address cannot be changed for security reasons
  • Profile changes take effect immediately
Password Security
  • Change password regularly for security
  • Must be 8+ characters with uppercase, lowercase, and numbers
  • New password must be different from current one
  • You'll be logged out after password changes
Security: Your data is private and secure. Only you can access your collections and resources.

Common Issues

Can't save changes?

  • Check internet connection
  • Ensure all required fields are filled
  • Try refreshing the page and logging in again

Search not working?

  • Check spelling and try different keywords
  • Use partial words or shorter terms
  • Clear filters and try again
Pro Tips

Productivity Hacks:

  • Use keyboard shortcuts (Ctrl+Click to open links in new tabs)
  • Bookmark frequently used collections
  • Set up collections for different learning goals
  • Review your "In Progress" resources weekly

Organization Strategy:

  • Create a "Quick Reference" collection for documentation
  • Use descriptive titles that you'll remember later
  • Tag resources with year or difficulty level in titles
Daily Tip

Stay Organized: Review your "In Progress" resources weekly and update their status. This helps you track your learning momentum and identify resources that might need more attention.

Use the search feature to quickly find resources by title, author, or status.
