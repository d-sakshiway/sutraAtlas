To create a new collection:

  1. Go to the Collections page from your profile
  2. Click the "Create New Collection" button
  3. Enter a name and optional description
  4. Click "Create Collection" to save

To add resources:

  1. Open the collection you want to add to
  2. Click the "Add Resource" button
  3. Fill in the title (required) and other details
  4. Set the status and add a URL if available
  5. Click "Add Resource" to save

  • Not Started - You haven't begun this resource yet
  • In Progress - You're currently working through this resource
  • Paused - You've temporarily stopped working on this
  • Completed - You've finished this resource

To update your profile:

  1. Go to your Profile page
  2. Click "Edit Profile"
  3. Update your username (email cannot be changed)
  4. Click "Save Changes"
